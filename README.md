## 概要
学生寮に住んでいる学生たちが持ち物を売買や交換するためのオンラインプラットフォーム
<br>
<br>

## 目的
学生たちの経済的負担を軽減し、持ち物のリサイクルと再利用を促進すること
<br>
<br>

## 特徴
- 学生に特化したマーケットプレイス
- 写真のアップロードとプレビュー
- カテゴリー分類と検索
- ページネーション
- コメント機能
- コミュニティ
- いいね機能
<br>
