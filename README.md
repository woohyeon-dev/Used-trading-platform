## 📝 About The Project
学生の経済的負担を軽減し、持ち物のリサイクルと再利用を促進するために作りました。<br>
<br>

## ☁️ Overview
学生寮に住んでいる学生たちが持ち物を売買や交換するためのオンラインプラットフォーム<br>
<br>

- 学生に特化したマーケットプレイス
- 写真のアップロードとプレビュー
- カテゴリー分類と検索
